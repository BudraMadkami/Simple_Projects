# **CRASHE/SHORT COURSE ON GIT :**

1. **Install Git from the Website** [**https://git-scm.com/install/windows**](fromgit)**(for Windows or Mac )..**
2. **Create one folder in the Desktop (ex. My Fold)**
3. **Open Git**
4. **Name Set up (git coffin --global user. name "Your Name" )**
5. **E-mail setup (git coffin --global user.email "xyz@gmail.com" )**
6. **To access the folder created in git type the command (Cd <your folder name> /   example: cd BM/ )**
7. **To access file inside the folder type command( cd <your file name > / )**
8. **To add your file through Git, type ( git add <yourfilename.txt> )**
9. **To check whether it is synced in or not ( git status )->you will see the master file**
10. **To work on it, type ( git commit -m "I'm working now" )**
11. **To check whether it is working or not (git status ) -> will show working on ..**
12. **Now change something inside your file and check(git status )  status you will see modified. To access what actually had been modified ( git add <your file name>.txt) and (git status )**
13. **To add all current directories (git add . ) -> (git status )**
14. **When you want to commit all changes everywhere, use (git add -A)/(--all)**
15. **When you want to update tracked files but ignore untracked/new ones, use(git add --u / git add --update)**
16. **to see detailed changes made by user (git log )- will be shown with (name ,date and emaial )**
17. **To see inshort (git log --one line )**
18. **to see within 2 lines (git log -2 )**
19. **to unstage staged file (git restore --staged <file name .extension> )**
20. **to see what changes have done in the directory (git diff )-> it will show all the changes have done by the author or else .(Changes will be shown in green color.)**
21. **to track previous changes (git diff --staged <file name > )**
22. **(HEAD~2 = two commits before HEAD)**
23. **Shows what changed in the most recent commit. ( git diff HEAD~1 HEAD )**
